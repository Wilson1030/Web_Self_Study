HTML定义
    HTML: 超文本标记语言

标签语法:
    1. 标签成对出现, 中间包裹内容
    2. <>里面放英文字母(标签名)
    3. 结束标签比开始标签多一个/
    4. 双标签: 成对出现的标签
    5. 单标签: 只有开始标签, 没有结束标签
        例如: <br>: 换行  <hr>: 水平线

HTML基本骨架:
    <html>
        <head>
            <title>网页标题</title>
        <head>
        <body>
            网页主体
        </body>
    </html>
    html: 整个网页
    head: 网页头部, 存放给浏览器看的代码, 例如CSS
    body: 网页主体, 存放给用户看的代码, 例如: 图片, 文字
    title: 网页标题
    VSCode快速生成骨架: !(英文输入法)配合Enter或者Tab

标签的关系:
    作用: 明确代码的书写位置
    1. 父子关系(嵌套关系)
    2. 兄弟关系(并列关系)

注释:
    注释就是对代码的解释和说明, 其目的是让人们能够更加轻松的了解代码, 注释是编写程序时, 写程序的给一个语句, 程序段, 函数等的解释或提示, 能提高程序代码的可读性
    VSCode中添加/删除注释的快捷键:Ctrl + /

标题标签:
    一般用在新闻标题, 文章标题, 网页区域名称, 产品名称等等
    标签名: h1-h6 (双标签)

段落标签:
    一般用在新闻段落, 文章段落, 产品描述等等
    标签名: p (双标签)
    显示特点:
        1. 独占一行
        2. 段落之间存在间隙

换行与水平线标签:
    换行: <br> (单标签)
    水平线: <hr> (单标签)

文本格式化标签:
    作用: 为文本添加特殊格式, 以突出重点, 常见的文本格式: 加粗, 下划线, 倾斜, 删除线等
    标签格式:
        加粗: <strong></strong>   或者 <b></b>
        倾斜: <em></em>   或者 <i></i>
        下划线: <ins></ins>   或者 <u></u>
        删除线: <del></del>    或者 <s></s>
    Ps: 推荐左边格式, 标签自带强调含义

图像标签:
    作用: 在网页中插入图片
    标签格式:
        <img src="图片的URL">
        src用于指定图像的位置和名称, 是<img>的必须属性
    属性:
        1. alt 作用: 替换文本 (图片无法显示的时候显示的文字)
        2. title 作用: 提示文本 (鼠标悬停在图片上面的时候显示的文字)
        3. width 作用: 图片的宽度 (值为数字, 没有单位)
        4. height 作用: 图片的高度 (值为数字, 没有单位)
    写法:
        1. 属性名 = "属性值"
        2. 属性写在尖括号里面, 标签名后面, 标签名和属性之间用空格隔开, 不区分先后顺序

路径:
    路径指的是查找文件时, 从起点到终点经历的路线
    路径分类:
        1. 相对路径: 从当前文件出发查找目标文件
            /表示进入某个文件夹里面
            .表示当前文件所在文件夹
            ..表示当前文件所在的上一文件夹
        2. 绝对路径: 从盘符出发查找目标文件

超链接:
    作用: 点击跳转到其他页面
    标签格式: <a href="网站路径" target="_blank">名称</a>

音频标签:
    标签格式: 
        <audio src="音频的URL"></audio>
    属性:
        1. src(必须属性) 作用: 音频URL (支持格式: MP3, Ogg, Wav)
        2. controls 作用: 显示音频控制面板
        3. loop 作用: 循环播放
        4. autoplay 自动播放 (为了提升用户体验, 浏览器一般会禁止自动播放功能)

视频标签:
    标签格式:
        <video src="视频的URL"></video>
    属性:
        1. src(必须属性) 作用: 音频URL (支持格式: MP3, Ogg, Wav)
        2. controls 作用: 显示音频控制面板
        3. loop 作用: 循环播放
        4. muted 作用: 静音播放
        5. autoplay 自动播放 (为了提升用户体验, 浏览器一般会禁止自动播放功能)

网页制作思路:
    从上到下, 先整体再局部, 逐步分析制作
    分析内容, 写代码, 保存, 刷新浏览器, 看效果

列表:
    作用: 布局内容排列整齐的区域
    列表分类: 无序列表, 有序列表, 定义列表
        1. 无序列表:
            作用: 布局排列整齐的不需要规定顺序的区域
            标签: ul嵌套li, ul是无序列表, li是列表条目
            标签格式:
                <ul>
                    <li>第一项</li>
                    <li>第二项</li>
                    <li>第三项</li>
                    ...
                </ul>
            特点: 独占一行
            注意事项:
                1. ul标签里面只能包裹li标签
                2. li标签里面可以包裹任意内容
        2. 有序列表
            作用: 布局排列整齐的需要规定顺序的区域
            标签: ol嵌套li, ol是有序列表, li是列表条目
            标签格式:
                <ol>
                    <li>第一项</li>
                    <li>第二项</li>
                    <li>第三项</li>
                    ...
                </ol>
            注意事项:
                1. ol标签只能包裹li标签
                2. li标签里面可以包裹任意内容
        3. 定义列表
            标签格式:
                dl嵌套dt和dd, dl是定义列表, dt是定义列表的标题, dd是定义列表的描述/详情
                <dl>
                    <dt>列表标题</dt>
                    <dd>列表描述/详情</dd>
                    ...
                </dl>
            注意事项:
                1. dl里面只能包含dt和dl
                2. dt和dd里面可以包含任何内容

表格:
    标签格式: 
        table嵌套tr, tr嵌套tg/th
    说明:
        table 表格
        tr 行
        th 表头单元格
        td 内容单元格
    提示:
        在网页中, 表格默认没有边框线, 使用border属性可以为表格添加边框线
    表格结构标签:
        作用: 用表格结构标签把内容划分区域, 让表格结构更清晰, 语义更清晰
        属性:
            1. thead 表格头部 (表格头部内容)
            2. tbody 表格主体 (主要内部区域)
            3. tfoot 表格底部 (汇总信息区域)
    合并单元格:
        作用: 将多个单元格合并成一个单元格, 以合并同类信息
        合并单元格的步骤: 
            1. 明确合并的目标
            2. 保留最左最上的单元格, 添加属性(取值是数字, 表示需要合并的单元格数量)
                跨行合并, 保留最上单元格, 添加属性rowspan
                跨列合并, 保留最左单元格, 添加属性colspan
            3. 删除其他单元格

表单:
    作用: 收集用户信息
    使用场景:
        1. 登陆页面
        2. 注册页面
        3. 搜索区域
    input标签基本使用:
        input标签type属性值不同, 则功能不同
        标签格式:
            <input type="...">
        type属性:
            1. text 文本框, 用于输入单行文本
            2. password 密码框
            3. radio 单选框
            4. checkbox 多选框
            5. file 上传文件
    input标签占位文本:
        作用: 提示信息
        标签格式:
            <input type="..." placeholder="提示信息">
        注意: 用于文本框和密码框
    单选框radio:
        常用属性:
            1. name 控件名称 (控件分组, 同组只能选中一个(单选功能))
            2. checked 默认选中 (属性名和属性值相同, 简写为一个单词)
        标签格式:
            <input type="radio" name="..." checked>
    上传文件file:
        默认情况下, 文件上传表单控件只能上传一个文件, 添加multiple属性可以实现文件多选功能
        标签格式:
            <input type="file" multiple>
    多选框checkbox:
        多选框也叫复选框
        默认选中: checked
        标签格式:
            <input type="checkbox" checked>

下拉菜单:
    标签格式:
        select嵌套option, select是下拉菜单整体, option是下拉菜单的每一项, selected为默认选中
        <select>
            <option>...</option>
        </select>

文本域标签:
    作用: 多行输入文本的表单控件
    标签格式:
        textarea 双标签
        <textarea>提示文字</textarea>

label标签:
    作用: 网页中, 某个标签的说明文本
    经验呢: 用label标签绑定文字和表单控件的关系, 增大表单控件的点击范围
    标签格式1:
        1. label标签只包裹内容, 不包裹表单控件
        2. 设置label标签的for属性值和表单控件的id属性值相同
        <input type="radio" id="...">
        <label for="..."></label>
    标签格式2:
        使用label标签包裹文字和表单控件, 不需要属性
        <label><input type="radio">...</label>

按钮:
    标签格式:
        <button type="">按钮</button>
    常用属性:
        1. submit 提交按钮 (点击后可以提交数据给后台)(默认功能)
        2. reset 重置按钮 (点击后将表单控件恢复默认值)
        3. button 普通按钮 (默认没有功能, 一般配合JavaScript使用) 
    注意: 需要form标签统一包裹

无语义的布局标签:
    作用: 布局网页 (划分网页区域, 摆放内容)
        1. div: 独占一行
        2. span: 不换行
    标签格式:
        <div>div标签, 独占一行</div>
        <span>span标签, 不换行</span>

字符实体:
    作用: 在网页中显示预留字符
    格式:
        空格 &nbsp;
        小于号(<) &lt;
        大于号(>) &gt;
